## GitHub上传代码的命令行,作为备份
* 把项目先git clone下来
* 在文件夹中进行修改
* 在文件夹中打开终端
* git init
* git add .
* git commit -m "commit"
* git push -u origin master
* 输入用户名和密码
* 完成上传<br>
可见相关的教程：
[Linux](https://blog.csdn.net/daydayjump/article/details/80850390 "Ubuntu") 
[Windows](https://www.cnblogs.com/cxk1995/p/5800196.html "Win10")
